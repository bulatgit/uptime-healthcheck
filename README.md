# Website Uptime Probe

This repository provides a lightweight, on-demand network availability probe
implemented using GitHub Actions.

The probe is designed for short-lived connectivity validation without requiring
persistent infrastructure or external monitoring services.

## Features

- Manual execution
- Configurable probe duration
- Minimal runner footprint
- Automatic cleanup after completion

## Usage

1. Open the **Actions** tab
2. Select **Network Uptime Probe**
3. Click **Run workflow**
4. Set the desired probe duration

The probe performs availability checks and exits automatically.
